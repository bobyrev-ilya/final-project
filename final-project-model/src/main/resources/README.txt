Структура приложения (возможно кому-то поможет xD)

- В качестве слоя для взаимодействия пользователя с банкоматом выделил интерфейс ClientService. Для получения услуги
getBalance необходимо передать карту и ввести пин.

- Вызываемый клиентский запрос прокидывавется на слой сервисов банка (interface BankService), содержащий в себе
вспомогательные методы для взаимодействия с источником данных

- Поступающие запросы из сервиса банка обрабатываются CardService, который выступает сервисом для хранилища данных
о картах CardRepository
